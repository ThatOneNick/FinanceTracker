# Finance Tracker

A desktop application used for keeping track of income, bills, subscriptions, and other monthly expenses.

## Features

- Track income and displays total income
- Track bills and displays all bills total cost
- Track subscriptions and displays all subscriptions total cost
- Track other expenses and displays all other expenses total cost
- Monitor net income/cash flow.

## Installation Instructions

Click on the latest release on the right side of the page. 
Under "Assets" Click "FinanceTracker-v1.0.0.zip to download the zip file.
Once downloaded, right click on and unzip the file.
The app is now ready to use. Please note that the first time you launch the app windows may display a warning about running apps from an unknown source. Click on "More Info", then "Run Anyway". This warning should no longer pop up after the first time.
